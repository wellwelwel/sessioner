# Security Policy

## Is Sessioner Safe?

**Sessioner** is an open-source project, so you can see both the [Source Code on **GitHub** Repository](https://github.com/wellwelwel/sessioner) and the [Distribution Code on **NPM**](https://www.npmjs.com/package/sessioner?activeTab=code).

---

## Supported Versions

Currently, security updates will be applied to the following versions of **Sessioner**:

| Version | Supported          |
| ------- | ------------------ |
| 0.x.x   | :white_check_mark: |

---

## Reporting a Vulnerability

- Please, give detailed reports
- Include steps to reproduce the vulnerability, and if possible, a patch or workaround.
- Include the specific version of **Sessioner** you are using.

**Reporting:**

- https://github.com/wellwelwel/sessioner/security/advisories

> Once the issue has been resolved, you will be attributed a part of the report.
